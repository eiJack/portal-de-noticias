# Portal de Notícias

## Sobre o Projeto

O **Portal de Notícias** é um projeto acadêmico desenvolvido para a disciplina de **Encontrabilidade**, do **6º semestre do curso de Sistemas para Internet**.

O objetivo do portal é disponibilizar notícias organizadas por categorias, proporcionando uma navegação simples, intuitiva e eficiente para os usuários. A estrutura do projeto foi planejada considerando princípios de encontrabilidade, organização da informação e experiência do usuário.
